Домашнее задание по Git Bash 
1. Посмотреть где я - pwd
2. Создать папку - mk.dir group_23
3. Зайти в папку - cd group_23
4. Создать 3 папки - mkdir dir_1 dir_2 dir_3 или mkdir {00..3} 
5. Зайти в любую папку - cd dir_1
6. Создать 5 файлов (3txt 2json) - touch 3.txt 2.json
7. Создать 3 папки -  mkdir dir_1 dir_2 dir_3
8. Вывести список содержимого папки - ls -la
9. Открыть любой txt файл - vim 1.txt
10. Написать туда что-нибудь, любой текст - INSERT
11. Сохранить и выйти - Esc :wq
12. Выйти из папки на уровень выше - cd ..
13. Переместить любые 2 файла, которые вы создали, в любую другую папку - mv 1.txt dir_2/1.txt
14. Скопировать любые 2 файла, которые вы создали, в любую другую папку - cp 1.txt dir_2/1.txt
15. Найти файл по имени - find Alina.txt
16. Просмотреть содержимое в реальном времени (команда grep) изучите как она работает - grep -w "Домашнее задание" Alina.txt
17. Вывести несколько первых строк из текстового файла - head  -3 Alina.txt
18. Вывести несколько последних строк из текстового файла - tail Alina.txt
19. Просмотреть содержимое длинного файла (команда less) изучите как она работает - less Alina.txt
20. Вывести дату и время - date
